# Challenge Front-End

Proyecto de práctica Front-End enfocado en el desarrollo de una interfaz web
responsive utilizando únicamente **HTML y CSS**.

El objetivo principal del proyecto es poner en práctica conceptos fundamentales
del diseño web como:
- Maquetación con Flexbox
- Uso de etiquetas semánticas de HTML
- Estilos modernos con CSS
- Aplicación de tipografías externas
- Diseño responsive mediante media queries
- Organización visual y estructura del contenido

El proyecto consiste en una **card informativa** con texto, estadísticas e imagen,
adaptada para distintos tamaños de pantalla (desktop y mobile).

## Tecnologías utilizadas
- HTML5
- CSS3
